# Joints #1 + #2 collapse into one — the "no true zero" principle

**Date:** 2026-07-07 · **Status:** unifies two open joints into one contested question; does not resolve it · **Lit-checked first** (CLAUDE.md rule).

## The principle (Pitarn)

The framework's core carries **no exact zeros** — nothing reaches true 0, because an exact zero is absolute stasis and "activity in the universe would stop." Applied to the dark-energy floor: $V_{\min}$ approaches 0 but is never *exactly* 0.

## What it does to joint #2 (the Weinberg wall)

- LMU's $A$-field is a **quintessence** (dynamical) scalar, which is exactly the class that **evades Weinberg's no-go** — the no-go assumes a *constant* vacuum energy, and a rolling field is not constant. So the framework **never needs $V_{\min}=0$ exactly.** The "Weinberg wall" (why the floor is *exactly* zero) does not apply.
- **Honest limit:** quintessence evades the exact-0 no-go but does **not** solve the cosmological-constant problem — the *tiny value* ($\Lambda\!\sim\!10^{-122}$) is still unexplained, a puzzle every model shares. So joint #2 **downgrades** from "the Weinberg wall (exact 0)" to "the standard CC problem (tiny value)." Not solved; LMU no more exposed than anyone.

## The unification (the real content)

Apply "no true zero" to $V_{\min}$ ⇒ the endgame is a **tiny de Sitter** (small $H_\infty>0$), not exact Minkowski. Then:

- a tiny de Sitter has a Gibbons–Hawking temperature $T=H_\infty/2\pi>0$ — it is **not** truly dead;
- **Volovik 2023** (*Thermodynamics and Decay of de Sitter Vacuum*) argues de Sitter is **unstable**, $\Gamma>0$ — it decays / creates matter — which is **exactly the $P>0$ that joint #1 (relighting) needs.**

So one principle supplies **both** $V_{\min}\neq0$ (joint #2) **and** $\Gamma\neq0$ (joint #1). The two open joints **collapse into a single question:**

> **Is de Sitter stable ($\Gamma=0$, Boddy–Carroll–Pollack 2014) or unstable ($\Gamma>0$, Volovik 2020/2023)?**

If Volovik → **both joints close.** If Boddy–Carroll–Pollack → both stay open. Either way it is now **one** contested question, not two. The "no true zero" principle *picks the Volovik side* — a postulate, not a derivation.
