## กติกาการทำงาน
- "ตรวจ/verify" = รันสคริปต์ตรวจที่มีอยู่ก่อนเสมอ ห้ามเปิด audit เอกสารเองโดยไม่ถามสโคปกลับ
- ห้ามอ่านไฟล์ .tex ทั้งไฟล์ — อ่านเฉพาะช่วงบรรทัดที่ถูกชี้ (ใช้ grep/sed)
- หนึ่งงาน = หนึ่งสโคป จบแล้วหยุด ห้ามขยายขอบเขตเอง เปลี่ยนงานให้ /clear ก่อน
- workflow/subagent ต้องประกาศจำนวน agent สูงสุดก่อนเริม (เกิน 20 ต้องขออนมัติ)
- verify แบบ batch: verifier 1 ตัวต่อ findings ทั้งชุด ห้ามหนึง finding หนึ่ง agent
- ข้อโต้แย้งเชิงตัวเลขต้องแนบสคริปต์รันซ้ำได้ ไม่งั้นนับเป็น [Hypo]
- ก่อนคำนวณ/สร้างของใหม่ ค้นวรรณกรรมก่อนเสมอ (WebSearch/Consensus) ว่ามีคนทำแล้วยัง — รายงานเจ้าของก่อน แล้วค่อยตัดสินว่ายืน/ต่อยอด/เทียบ ห้าม reinvent (เปลืองเวลา+โทเคน; ผูกกับ R7: กลไก=ของเจ้าของ ของเรา=แค่การต่อ)
- ห้ามอวย: ไม่มีประโยคชม/แสดงความยินดี/พิธีปิดท้ายที่เล็งตัวบุคคล ทั้งในแชทและเอกสาร — แชทคุยงาน = แห้ง ตรง ไม่มีพิธี; เอกสารภาษาพูดใส่ภาพ/สีสันของเนื้อหาได้เต็มที่ แต่ศูนย์คำเยินยอ (ประโยคที่หน้าที่เดียวคือลูบหัวคนอ่านหรือคนเขียน = ตัดทิ้ง)
- **ปมเปิดของ LMU มีแค่ 2 (settled 2026-07-07):** (1) "+3 spread" [soft], (2) wiring ω=inflaton [Hypo]. **de Sitter stability / swampland / P>0 / V_min=0 = คำถาม field-wide ที่ LMU แชร์ ไม่ใช่ปมเฉพาะ/ตัวปิดวงของ LMU** เพราะ reset เป็น deterministic flash (survivor ระเหยแน่นอน→แฟลชเขี่ยสนามบนที่ราบ α-attractor) ไม่ใช่ probabilistic decay. **ห้ามยกกลับมาลิสต์เป็นปม LMU อีก.** ถ้าเจอข้อความเก่าในไฟล์ (`.tex` body, review docs เก่า, revision entries) ที่พูดว่า de Sitter/P>0/Minkowski = ปมของ LMU → นั่นคือ **ชั้น superseded** ห้ามเชื่อ/ยกกลับ ให้ชี้ไป `review/LMU_SYNTHESIS_2026-07-07.md` (= single source of truth ของสถานะเปิด) แล้วรายงานว่าไฟล์นั้นยังไม่ sync ไม่ใช่เปิดประเด็นใหม่
- **นิยาม "+3 spread" ปักหมุดแล้ว (2026-07-07, อย่าให้เด้ง):** ปมนี้ = **residual ENERGY** ของ aeon เก่าต้อง redshift ต่ำกว่า **vacuum floor ที่วัดได้** Λ/M_Pl⁴≈10⁻¹²² ภายใน N≈57 e-folds (ที่ CMB ล็อก) → ต้องการ effective exponent **k≈5** ซึ่งเป็น **GR redshift จริงตามองค์ประกอบ** (curvature a⁻² ช้า, radiation a⁻⁴, shear a⁻⁶ เร็ว; พลังงานหายเข้า expansion, Wald 1984) **ไม่ใช่ "spread สถิติ" (√patches)**. คำถามเปิด = ก้อน L0 "สงบ/low-Weyl" (curvature-หนัก, ช้า) พก shear/radiation พอไหม. **แยกจาก** observable smoothness = density contrast 10⁻⁵ (ทำง่าย, คนละปริมาณ). 10⁻¹²² = **ค่าวัดได้ปัจจุบัน (Λ คงที่)** ห้ามเรียกว่า "ปลายยุคไกล" หรือ "ผิดหน่วย". สคริปต์ยืน: `code/residual_dilution_to_floor.py`
